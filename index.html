Setup Guide — How to get your credentials
Follow these steps if you haven't set up your SP-API app yet
1

Register as an Amazon Developer

Go to developer.amazonservices.com and sign in with your Seller Central account. Accept the developer agreement.

2

Get LWA Credentials

Go to developer.amazon.com → Login With Amazon. Create a security profile. Copy the Client ID and Client Secret.

3

Create SP-API App & Get Refresh Token

In Seller Central go to Apps & Services → Develop Apps. Create a new app. Then authorize it to get your Refresh Token (starts with "Atzr|").

4

Create AWS IAM User

Go to AWS IAM Console. Create a user, attach the AmazonSellerPartnerAPIRole policy. Generate access keys and copy them here.

5

Save & Test

Fill in all fields above, click Save Credentials, then click Test Connection. All 4 status indicators should turn green!
